# Fleur Animée avec GSAP et ScrollTrigger

Ce projet montre comment faire **pousser une fleur SVG** au scroll et ajouter des animations de pigeons, sans modifier le SVG d’origine.

---

## Installation

1. **Cloner le projet :**
```bash
git clone https://github.com/StepCode3630/flower-scroll.git
cd <flower-scroll>
```
2. **Installer les dépendances:**
```bash
npm install
```
3. **Lancer le server:**
```bash
npm run dev
```
4. **Ouvrir dans le naviguateur:**
   Par défault, sur http://localhost:5173
